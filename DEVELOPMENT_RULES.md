# 開發規範與協作規則 (Development Rules & Guidelines)

此檔案紀錄了本專案開發過程中的核心規範，AI 助手在每次啟動或執行任務前必須主動讀取並嚴格遵守。

## 1. 程式碼撰寫授權規則
- **核心指令**：除非使用者明確輸入「**開始**」二字，否則 AI 助手不得自行撰寫、修改或刪除任何專案程式碼（包含 HTML, CSS, JS 等）。
- **非授權期間行為**：在尚未獲得「開始」指令前，AI 僅能進行以下活動：
    - 需求分析與討論。
    - 程式碼邏輯研究與解釋。
    - 制定實作計畫（Implementation Plan）。
    - 回答使用者的提問。

## 2. 專案背景與技術棧
- **專案名稱**：暗棋 (Dark Chess)
- **技術棧**：原生 HTML / CSS / JavaScript (Vanilla JS)。
- **風格要求**：高質感的現代網頁設計，包含動態效果與細膩的視覺美感。

## 3. 持續更新
- 當有新的協作規範或開發偏好時，應同步更新於此檔案中。
